# IP Info
A simple project using Vite + React
## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/hendrahend/ip-info.git
   ```
2. Move to project directory
   ```sh
   cd ip-info
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

## Usage

Run the project
   ```sh
   npm run dev
   ```
